Official Formbar User Manual

Here you can find everything necessary to understanding the formbar, this is a much more siplified version of our github wiki, we hope you enjoy using the formbar.

Students

Thumbspanel

The thumbspanel will allow you to answer polls which are started by the teacher. You can also ask for help or to go to the bathroom from here.

Types Of Polls

The first time of poll, a thumbs poll, is answered with the up, down, and wiggle buttons which simulates the way a person moves their thumb in respons to a question. Upon hitting one on these you wil be prompted with a message telling you "thank you for your answer" or "you already submitted this answer."

The next is a 4 answer poll which uses the A, B, C, and D buttons, this is typically a multiple choice question. Clicking these when an ABCD poll is active will result in the same message that the up, down, and wiggle gave.

The final type is an essay button which will take any response you input. This is of course for open ended questions, and more personal surveys.

Other Thumbspanel Features

By entering either the bathroom break or help sections, with a simple click you can let the teacher know you need help, or that you really gotta go to the bathroom. Remember if you send a help ticket, keep working out the issue until the teacher comes to your aid.

Chat

The chat is a place where students and teachers can interact, it allows for communication amongst all logged in users. Just enter your message and hit the enter key or click the send button and your message will be sent to all others in the class.

There is also edit and delete functions represented by the blue pencil and red wastebin. Each will display a prompt asking you to confirm the action.

What Are We Doing?

This will display the lesson plans for that day, it will need to be manually updated every time a new session begins.

Formbar

Background Music and Sound Effects

Both of these tabs display a list of sounds you can play. Background music is a collection songs added by students or teachers overtime. Sound effects is a collection of non-looping sounds you can play.

Advanced Homepage/Advanced Mode

Not reccomended for those who don't understand the formbar well, it'll allow you to do everything the normal formbar can in a much more compact interface. It's a lot more technical than the normal layout and, once again, not reccomended for new users.

Set Default Homepage

This will allow you to change the type of page you user, the default, advanced, or quick pages are dependant on user preference, we reccomend sticking with default until you understand the formbar completely.

VirtualBar

The virtualbar will allow you to view a visual list of all poll answers, it will not display names.

Games

The game section is filled with fun little mini-games you can play on your downtime. These were all made from scratch and some even include fun trivia about the class!

Leaderboards

The Leaderboard displays all the scores for each game and ranks them from best to worst. By hitting the green arrows you can look at each individual game.

Teacher

Teacher Controls

The control panel is where teachers can view a list of all students, change user permissions, manage user profiles, view poll history and help/bathroom tickets, and full reboot the formbar.

User Accounts

This will allow teachers to change a students password per their request, delete their accounts, and reward them with digipogs. You just have to click the corresponding link and it'll guide you from there.

Changing Passwords

To change a user's password, after the input box comes up, you will enter the desired new password and hit "OK" have the student login once again.

Adding and Removing Digipogs

To add digipogs simply enter a static whole number and hit "OK" to remove them put a minus sign before the number to make a negative and then hit "OK".

Other Documents

If you're looking for a more technical user guide, we have one on our github page which you can access here, you can also find developer contact info on GitHub in case you cannot figure something out.
